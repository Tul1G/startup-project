# Исследование рынка стартапов и венчурных инвестиций
Описание проекта

Проект посвящён исследованию рынка стартапов и венчурного финансирования на основе исторических данных о компаниях, инвестиционных раундах и возврате средств инвесторам.

Цель проекта — выявить закономерности финансирования стартапов, исследовать динамику инвестиционного рынка и определить наиболее перспективные направления для инвестирования с точки зрения 2015 года.

В рамках проекта были выполнены предобработка и исследовательский анализ данных (EDA), анализ выбросов, сегментация рынка, исследование инвестиционных раундов и динамики возврата средств, а также визуализация результатов в Jupyter Notebook.

⸻

Цели исследования
 - Исследовать структуру и динамику венчурного финансирования.
 - Определить наиболее перспективные сегменты рынка.
 - Проанализировать объёмы и популярность различных типов инвестиций.
 - Выявить типичные и аномальные значения финансирования.
 - Исследовать динамику инвестиционной активности по годам.
 - Оценить устойчивость возврата средств для различных типов финансирования.
 - Подготовить рекомендации по инвестиционной стратегии.

Данные

В проекте использовались два датасета:

cb_investments

Содержит информацию о стартапах:
 - отрасли и сегменты рынка;
 - объёмы инвестиций;
 - количество инвестиционных раундов;
 - типы финансирования;
 - даты финансирования;
 - статус компаний;
 - география компаний.

cb_returns

Содержит информацию о возврате средств инвесторам:
 - объёмы возвратов по годам;
 - данные по различным типам финансирования.

Что было сделано

Предобработка данных
 - Проверка структуры и типов данных.
 - Обработка пропусков и дубликатов.
 - Приведение числовых и временных данных к корректным форматам.
 - Очистка столбца funding_total_usd.
 - Восстановление пропущенных значений дат финансирования.
 - Оценка полноты данных после очистки.

Исследовательский анализ данных (EDA)
 - Анализ распределения объёмов финансирования.
 - Исследование количества и структуры инвестиционных раундов.
 - Анализ популярных отраслей и сегментов рынка.
 - Исследование динамики инвестиционной активности по годам.
 - Анализ объёмов и популярности различных типов финансирования.

Инжиниринг признаков
 - Сегментация компаний по сроку финансирования.
 - Классификация рынков на массовые, средние и нишевые сегменты.
 - Объединение малых сегментов в категории mid и niche.

Работа с выбросами
 - Выявление аномальных значений финансирования методом IQR.
 - Анализ выбросов отдельно для разных сегментов рынка.
 - Исключение аномальных компаний из дальнейшего анализа.

Анализ динамики
 - Исследование изменения среднего размера инвестиционных раундов по годам.
 - Анализ активности инвестиционного рынка.
 - Исследование роста финансирования в массовых сегментах.
 - Анализ динамики возврата средств по типам финансирования.

Визуализация данных
 - Построение графиков распределения финансирования.
 - Визуализация динамики инвестиций и раундов.
 - Анализ трендов возврата средств.
 - Исследование сегментов рынка с помощью визуализации.

⸻

Ключевые результаты
 - Выявлены сегменты рынка с наиболее устойчивым ростом финансирования.
 - Определены наиболее популярные и наиболее капиталоёмкие типы инвестиций.
 - Обнаружены аномальные объёмы финансирования и оценено их влияние на рынок.
 - Исследована динамика инвестиционной активности и изменения рынка по годам.
 - Проанализирована устойчивость возврата средств для различных типов финансирования.
 - Подготовлены рекомендации по перспективным направлениям инвестирования.
